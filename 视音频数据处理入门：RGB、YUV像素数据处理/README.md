* 本文分别介绍如下几个RGB/YUV视频像素数据处理函数：
    * 分离YUV420P像素数据中的Y、U、V分量
    * 分离YUV444P像素数据中的Y、U、V分量
    * 将YUV420P像素数据去掉颜色（变成灰度图）
    * 将YUV420P像素数据的亮度减半
    * 将YUV420P像素数据的周围加上边框
    * 生成YUV420P格式的灰阶测试图
    * 计算两个YUV420P像素数据的PSNR
    * 分离RGB24像素数据中的R、G、B分量
    * 将RGB24格式像素数据封装为BMP图像
    * 将RGB24格式像素数据转换为YUV420P格式像素数据
    * 生成RGB24格式的彩条测试图
